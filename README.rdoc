== README

* use ruby 2.1.5

* clone the project

* cd movies-location-api

* run bundle install

* add database.yml - take help from database.yml.example in config folder

* add secrets.yml - take help from secrets.yml.example in config folder

* run rake db:create

* run rake db:migrate

* run rake load_data

* start server - run rails server

open localhost:3000 in browser
